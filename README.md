# やしきんのマイページリポジトリ

Next.jsを使用して作成したモダンなプロフィールページです。

## 特徴

- Next.js 16とApp Routerを使用
- Tailwind CSS v4でスタイリング
- TypeScript対応
- レスポンシブデザイン
- ダークモード対応

## セットアップ

### 必要なもの

- Node.js 20以上
- npm

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

### ビルド

```bash
npm run build
```

### 本番環境での起動

```bash
npm run start
```
